# 0.サーバーの趣旨
このサーバーはまったり建築鯖的生活鯖です。
# 1.管理者の能力の制限
管理者はプレイヤーに対しルールに基づく処分しか行うことができない。ただし、**軽微な処分**については独断で行うことができる。

## 1.1 基本的事項
この文章が何よりも偉いので、管理者たりともこれに反する行為を行うことができない。

## 1.2 補足

# 2.やってはいけないこととそれに対する処分

## 2.1 罰とは
重い順です。
プレイヤーに対する罰
|名称|概要|
|---|---|
|永久Ban|ただのBanです|
|期限付きBan|一定期間経過後に解除されるBanです。解除を忘れることがあると思うのでその時は異議申し立てを。|
|Kick|鯖から蹴られます。だいたいただの警告です。累積はありません。|
|Warn|警告ちょっと軽め。累積はありません。|

建築物/設備に対する処分
管理者はなんらかの不都合を引き起こす設備や建築などに対してのみこれを実行できる。
|名称|概要|
|---|---|
|強制撤去|運営による撤去です|
|撤去命令|期限(1~4週間まで設定可能。)までにプレイヤーが撤去/改善しなければならない。期限までに撤去/改善されなければ強制的に撤去される|
|警告|看板が貼り付けられます。|

## 2.2 やってはいけない行為と処分のリスト
|行為|プレイヤーに対する罰(Max)|ものに対する罰(Max)|
|---|---|---|
|いわゆる荒らし行為|永久Ban|-|
|他プレイヤーのプレイの妨害(ひどいものは荒らし行為として扱われます。)|期限付きBan|-|
|意図せずサーバーに大きな負荷をかける行為(意図していたと認定された場合は場合は荒らし行為として扱われます)|期限付きBan|強制撤去|
|周囲の景観とマッチしない建築を建てる事|Warn|撤去命令|
|工業拠点としての性格が強いものを全員からアクセスできない状態にすること|Warn|警告|
|大規模な環境破壊(定義は[チュートリアル](/tutorial.md)を見てください)|Kick|-|
|暴言/スパム|期限付きBan|-|
|チートの使用|永久Ban|-|

## 2.3 異議申し立て
管理者がルールに基づかない、または判例法主義に反するような処分を行った場合は、根拠と共にPublicな場(Discordなど)で異議申し立てをすること。
管理者はそれに対する対応を行う必要があり、拒否する場合は理由を述べる義務がある。

## 2.4 処分の公開
管理者は処分の詳細を平等、公平のために公開する義務がある。
[処罰履歴](処罰履歴.md)

# 3.参加者が心がけるべき事項
建築を建てるときは周囲の景観との調和を考えること。  
大自然を保護することを心がけ、身近な小さな自然を破壊し建築を建てること。
他の参加者に対し友好的な態度で接し、Modedマルチプレイコミュニティの発展に寄与すること。
Minecraftには重力が存在するという暗示にかかること。


# 4.意思決定の手法
基本的には意見を募り、それを基に意思決定を行うが、サーバー運営の都合上強権を行使することが管理者には許可されている、ただし乱発することは許可されない。
このリポジトリのIssueに投げられたら対応します。

# 4.1 このルールの改正
緊急性を要しない改正はCommitから三日後に施行される。  
緊急性を要する場合はCommitMessageの頭に[緊急\]と付ける。その場合はCommitから3分後に施行される。
